Function template random

boost::unit_test::data::random — Returns an infinite sequence of random numbers, in the given interval.

Synopsis

// In header: <boost/test/data/monomorphic/generators/random.hpp>


template<typename SampleType> 
  monomorphic::generated_by< monomorphic::random_t< SampleType > > 
  random(SampleType begin, SampleType end);

Description

The generator uses the default distribution, which is uniform (in [begin, end) for real numbers, and in [begin, end] for integers).
