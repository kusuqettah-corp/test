Portability concerns

Because the Boost Test Library is critical for porting and testing Boost libraries, it has been written to be conservative in its use of C++ features, and to keep dependencies to a bare minimum.

Boost.Test supports all main Boost compilers and platforms. Confirmation of its status on core and additional platforms/compilers can be seen by viewing Boost.Test's own internal regression test results on master branch status page or development branch status page
