BOOST_TEST_PASSPOINT

The macro BOOST_TEST_PASSPOINT is intended to be used to inject an &quot;unnamed&quot; checkpoint position. The macro signature is as follows: 

BOOST_TEST_PASSPOINT();

Unlike the macro BOOST_TEST_CHECKPOINT this macro doesn't require any message to be supplied with it. It's just a simple "been there" marker that records file name and line number code passes through.

Table 52. BOOST_TEST_PASSPOINT usage

Code

Output 

#define BOOST_TEST_MODULE example
#include <boost/test/included/unit_test.hpp>

//____________________________________________________________________________//

BOOST_AUTO_TEST_CASE( test_case )
{
    int* p = 0;

    BOOST_TEST_PASSPOINT();
    ++p;

    BOOST_TEST_PASSPOINT();
    ++p;

    BOOST_TEST_PASSPOINT();
    int j = *p;
}

//____________________________________________________________________________//
