Support of the auto-linking feature

For the Microsoft family of compilers the PEM provides an ability to automatically select proper library name and add it to the list of objects to be linked with. To employ this feature you required to include either header 

#include <boost/test/prg_exec_monitor.hpp>

or header 

#include <boost/test/included/prg_exec_monitor.hpp>

By default the feature is enabled. To disable it you have to define the flag BOOST_TEST_NO_LIB.
