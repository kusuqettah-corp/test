Testing tools

The Minimal Testing Facility supplies following four assertions: 

BOOST_CHECK(predicate)
BOOST_REQUIRE(predicate)
BOOST_ERROR(message)
BOOST_FAIL(message)

Their behaviour is modelled after the similarly named tools supplied by the UTF.
