Testing tools

The Minimal Testing Facility supplies following four tools: 

BOOST_CHECK(predicate)

BOOST_REQUIRE(predicate)

BOOST_ERROR(message)

BOOST_FAIL(message)

Their behavior is modeled after the <link linkend="utf.user-guide.testing-tools.reference">similarly named tools</link> implemented by the Unit Test Framework.
