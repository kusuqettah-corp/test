Usage

The Minimal Testing Facility is implemented inline in one header: 

#include <boost/test/minimal.hpp>

There are no special compilation instructions for this component.

The second and last change you need to make to integrate your test module with Minimal Testing Facility is the signature of your function main(). It should look like this: 

int test_main( int argc, char* argv[] )
{
  //...
}

Once you apply the changes, the test automatically starts running in monitored environment. Also you can start using testing tools provided by the Minimal Testing Facility and get uniform errors reporting.
